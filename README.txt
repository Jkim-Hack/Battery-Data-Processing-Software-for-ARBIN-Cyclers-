About Battery Data Processing Software (Dedicated to ARBIN Cycler)

This program uses the ARBIN Cylcer data and graphs it based on the user's parameters.


Developers: Main Dev: John Kim - johnkim1108@gmail.com
 	    Helper: Chris Yao - chrisyao2@gmail.com


MAKE SURE YOU DOWNLOAD JAVA: https://java.com/en/download/

API used: Apache POI (see LICENSE.txt) - https://poi.apache.org/
	  monitorjbl's "Excel-Streaming-Reader" - https://github.com/monitorjbl/excel-streaming-reader
	  kerner1000's "Javafx-Chart-Zooming" - https://github.com/kerner1000/javafx-chart-zooming
	  JavaFX - docs.oracle.com/javafx/
		  
		  
		
---------- SYSTEM REQUIREMENTS ----------

OS : Windows Vista
CPU: Intel Pentium or better.
RAM: 3GB of free RAM (At least for the largest data)
Drive Space: About 110mb of free space
